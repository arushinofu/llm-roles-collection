# AI Prompts Library

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://github.com/arushinofu/llm-roles-collection/blob/main/LICENSE.md)
[![Markdown](https://img.shields.io/badge/markdown-100000?style=flat&logo=markdown)](https://www.markdownguide.org)

Коллекция инструкций и ролей для настройки больших языковых моделей (LLM) под различные бизнес-задач и не только.

## О проекте

Репозиторий содержит промпты для ИИ-ассистентов: бизнес-аналитиков, маркетологов, разработчиков и других ролей. Каждая инструкция описывает:
- Цель и контекст работы
- Ограничения и правила
- Формат взаимодействия с пользователем

> 🔄 Структура репозитория, промты находится в разработке и будут дополняться.

## Как использовать

1. Найдите подходящий файл с инструкцией
2. Скопируйте его содержимое
3. Вставьте в чат с ИИ как системный промпт / первое сообщение
4. Следуйте сценарию инструкции
